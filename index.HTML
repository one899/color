FAQ

Got Questions?

Does Color Archive remove cloud control?
ColorOS cloud control is distributed across multiple modules. Retaining it helps with future feature compatibility.
Can I flash via TWRP?
No. A dedicated method is required. We recommend using BzdaBox for flashing.
How long until I receive the package?
Timing varies. The package will be sent to your email once ready. Please be patient.
Are future updates free?
Yes. Minor version updates (e.g. 16.0.1 → 16.0.3) require re-submission.
Can I relock the Bootloader?
No. Color Archive is a third-party system. Relocking will prevent booting.
Can I install official OTA updates?
Yes. Download the official ISO → Extract → Run coloros_install.sh → Reboot.
